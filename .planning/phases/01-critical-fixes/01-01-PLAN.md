# Phase 1 / Plan 01-01 — Critical Fixes

**Goal:** Fix every correctness, data-integrity, and high-severity security bug that would derail real use. First-pass drafted by local model (`gemma4:26b`), gated/rewritten by orchestrator, atomic-committed per fix.

## Tasks

### Data integrity / ingestion (STAB)
- [ ] T1 `STAB-05` — `templates/upload.html`: read `job.error` (not `job.error_msg`) so ingestion failures show the real cause.
- [ ] T2 `STAB-02` — `worker.py` `reap_stale_jobs`: only reap jobs with no live RQ job (check registry / `rq_job_id`), not blanket queued/running.
- [ ] T3 `STAB-01` — `ingest_async.py` `run_ingestion_job`: write `Document.chunks`, job `status='complete'`, `completed_at` to Postgres at end-of-job (worker owns truth, not the poller). Add a session helper.
- [ ] T4 `STAB-03`+`STAB-04` — store a stable unique `doc_id_prefix` (`doc_{id}`) on `Document` (new column, models.py) set at creation (app_fastapi.py); use it for upsert + delete; replace `abs(hash())` point IDs with a stable sha1-based int (ingest_async.py).
- [ ] T5 `STAB-06` — `app_fastapi.py` `run_inline`: open a fresh `SessionLocal()` and re-fetch `doc`/`job_record` by id inside the task.
- [ ] T6 `STAB-07` — `ingest_async.py`: derive `EMBED_DIM` from `embedder.get_sentence_embedding_dimension()` at collection creation; cache the worker embedder at module level + resolve device from DB config.
- [ ] T7 `STAB-08` — `llm_provider.py` anthropic empty-content guard + generic empty-shape error; `app_fastapi.py` `update_settings` null-email guard.

### Security (SEC)
- [ ] T8 `SEC-01` — `ingest.py`: add an SSRF guard (resolve host, reject loopback/private/link-local/reserved IPs + internal service names) applied in `_fetch_page`, `ingest_url`, crawl links, robots fetch; disable/validate redirects.
- [ ] T9 `SEC-02` — `app_fastapi.py` `register`: refuse to grant admin once an admin exists (and don't let open registration mint a second admin).
- [ ] T10 `SEC-03` — `app_fastapi.py` CORS: `allow_credentials=False` (Bearer-only).
- [ ] T11 `SEC-04` — `app_fastapi.py` `add_source`: enforce a max upload size; default filename when `secure_filename` empty; validate `doc_type` against extension.
- [ ] T12 `SEC-05` — `app_fastapi.py`/`llm_provider.py`/`ingest.py`: log exception detail server-side, return generic client messages (stop `detail=str(e)` leakage).

### Infra
- [ ] T13 `four-embedder-copies-ram` — `Dockerfile`: Gunicorn workers 4 → 2.
